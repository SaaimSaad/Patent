# 🚀 Cropped Delta Wing for VTOL Drone

## 📌 Overview

This repository contains the patent certification and research documentation for an innovative **Cropped Delta Wing Design** developed for **VTOL (Vertical Take-Off and Landing) Drone Systems**.

The project focuses on aerodynamic efficiency, structural optimization, and enhanced stability for fixed-wing VTOL drone applications.
